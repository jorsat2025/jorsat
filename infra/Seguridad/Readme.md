# Volder Honeypots

Infraestructura honeypot orientada a captura y análisis de amenazas.

## Objetivos

- captura de ataques
- análisis de comportamiento
- threat intelligence
- recolección de IOC
- integración con SIEM

---

## Honeypots

- Cowrie
- T-Pot
- SSH
- FTP
- Telnet
- Web
- Malware

---

## Funcionalidades

- captura de credenciales
- análisis de payloads
- logs centralizados
- integración con Suricata
- exportación de eventos
